## 説明
トップをねらってください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&ROM=https://github.com/CoBinee/acegun-msx/raw/main/rom/ACEGUN.ROM)

## 操作
- `←`<br>反時計回りにビームを回転移動
- `→`<br>時計回りにビームを回転移動
- ジョイスティックにも対応

## 動作環境
- 16KB 以上の RAM を搭載した MSX
